# Мебельный магазин

Проект для 3D просмотра мебели с возможностью настройки материалов.

## Технологии
- React
- TypeScript
- WebStorm
9. Мебельный магазин
   В приложении отображается каталог готовой мебели с возможность собрать свою
   через конструктор. На странице каталога должна быть возможность фильтровать
   мебель по:
- типу (стол, диван, шкаф)
- цвету
- размеру
  На странице отдельной позиции каталога:
- подробное описание модели
- галерея фотографий
- кнопка вызова виджета конструктора мебели. При ее нажатии в редактор
  подружается текущая модель
  Виджет конструктора мебели:
- позволяет менять размеры элементов мебели
- позволяет менять материалы, цвет отдельных элементов мебели
- в случае шкафа позволяет менять положение полок, удалять или добавлять их
- должен быть визуальный предпросмотр
  Сохранение на сервер: сконструированная модель мебели
